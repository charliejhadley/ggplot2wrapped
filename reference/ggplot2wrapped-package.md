# ggplot2wrapped: Your Year in ggplot2! Plus Tools For Exploring Your Dataviz With ggplot2!

This package is intended to provide a Spotify Wrapped experience for
ggplot2 users! There are interactive reports for summarising your usage
of geoms through the year and tools for analysing your code files
throigh the use of abstract syntax trees with the astgrepr package.
