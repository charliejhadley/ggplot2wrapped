# Core geoms dataset

A dataset containing all geoms from the ggplot2, ggrepel and ggtext
packages.

## Usage

``` r
data_geoms
```

## Format

### `data_geoms`

A data frame with 57 rows and 4 columns:

- geom_name:

  Name of geom

- package_name:

  Name of package containing geom

- docs_title:

  Title of geom documentation page

- docs_description:

  Description of geom documentation page
